    • Hi: this is a guide for you to make a good Graduation book
    you have to read About PDF and to build your Project on System Development life cycle (SDLC).
    there are six chapters in the Graduation Book.

    • Chapter One: Contain The Book Introduction
    • Chapter Two: Contain The Project Planning, Analysis
    • Chapter Three: Contain The Project Designs, Code
    • Chapter Four: Contain Integration, Testing
    • Chapter Five: Contain Acceptance, installation, deployment
    • Chapter Six: Contain The Recommendations, Conclusion, Sources
    When you Write the Chapter the Doctor will read it and give you feedback on to do it right.
